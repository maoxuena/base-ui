# Icon 图标

## 基础用法

:::demo 使用`color`、`size`属性来定义 Icon 的样式
icon/basic
:::

## Attributes

| 参数     | 说明   | 类型    | 可选值                                             | 默认值  |
| ------- | ------ | ------- | -------------------------------------------------- | ------- |
| size    | 尺寸   | number  |                                                    | 继承颜色 |
| color   | 颜色   | string  |                                                    | 继承字体大小 |
